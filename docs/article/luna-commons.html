  3. Get a free API Key at https://account.aliyun.com (opens new window)

# 各 Module 介绍

Module 名称 Module 介绍
luna-commons-ali 关于阿里oss和alpay支付宝的使用
luna-commons-common 基础工具包,加密字符等操作,http网络操作
luna-commons-api api集合模块,现阶段为学小易查题目接口,sm.ms 图床api (opens new window)
luna-commons-baidu 百度Api集合,包含人脸识别,人证审核百度地图等api封装
luna-commons-file 文件处理工具集合,包括ftp,httpd,fastdfs等文件操作
luna-commons-media 媒体流处理工具,包含ffmpeg工具封装,JavaCv图像处理

| luna-commons-message | 消息发送,短信邮件消息发送,有html模板附赠 HTML邮件模板 (opens new window) | luna-commons-tencent | 腾讯Api集合,包含人脸识别,人证审核腾讯地图等api封装 |
